Toetsenbord!

Met deze rekenmachine is het ook mogelijk je toetsenbord te gebruiken. Hier onder volgen de geaccepteerde toetsen:

  • Enter om de rekensom uit te voeren
  • Backspace om het laatst ingevoerde teken te verwijderen
  • Nummer toetsen om nummers in toe voeren

Helaas werken de /, *, - en + toetsen nog niet. Hier moet ik nog iets op verzinnen, maar wat ik tot nu toe heb geprobeerd heeft nog niet gewerkt.

Dit is een calculator van het beste soort. Gemaakt door mij en vrij om te gebruiken. Hoogstwaarschijnlijk heb ik iets meer vaardigheden zodra ik dit eenmaal werkend heb.

Deze calculator (ook wel rekenmachine genoemd) is gemaakt met HTML5, CSS3 en Javascript. Daarnaast gebruik ik Bootstrap voor de CSS indeling en jQuery voor javascript DOM taken. Dit is niet per sé noodzakelijk, maar maakt bepaalde taken wel een stuk makkelijker. Voor deze rekenmachine gebruik ik het Roboto font van Google. Misschien maak ik later een versie die niet afhankelijk is van externe libraries, maar misschien ook niet.
